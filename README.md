# 📚 Belajar HTML5 + CSS + JavaScript + PHP

Proyek ini merupakan website sederhana untuk pembelajaran dasar **Web Development** menggunakan **HTML5, CSS3, JavaScript, dan PHP**.  
Di dalam project ini terdapat berbagai elemen penting seperti struktur halaman web, styling modern, interaksi JavaScript, validasi form, dan koneksi database menggunakan PHP.

---

## 🚀 Fitur Utama

### 🌐 HTML5
- Struktur halaman menggunakan tag semantik:
  - `header`
  - `nav`
  - `main`
  - `section`
  - `footer`
- Form input data pengguna
- Tabel data
- Gambar dan daftar list

### 🎨 CSS3
- Desain modern dan responsive
- Gradient header
- Card layout
- Hover effect tombol & navbar
- Styling form dan tabel
- Warna menarik dan rapi

### ⚡ JavaScript
- Alert & Console Log
- Variabel dan tipe data
- Function biasa & Arrow Function
- Percabangan `if else`
- Perulangan `for`
- DOM Manipulation
- Inline Event
- Validasi Form Input

### 🐘 PHP
- Koneksi database menggunakan MySQL / PHPMyAdmin
- Pengolahan data form
- Menyimpan data ke database
