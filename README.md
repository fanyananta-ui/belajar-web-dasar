# 🎓 Website Profil & Form Pendaftaran - Teknik Informatika UMMI

Project ini merupakan tugas praktik mata kuliah **Pemrograman Web** yang menampilkan penerapan dasar **HTML5**, **CSS3**, dan **JavaScript** dalam pembuatan website sederhana namun interaktif.  
Website ini berisi halaman profil, tampilan modern, serta form pendaftaran dengan validasi input.

---

## 🚀 Fitur Utama

- **Struktur HTML5 Semantik**  
  Menggunakan elemen seperti `header`, `nav`, `main`, `section`, `table`, dan `footer`.

- **Desain Modern & Menarik**  
  Menggunakan warna gradasi, layout card, efek hover, serta tipografi yang rapi.

- **Formulir Interaktif**  
  Tersedia form pendaftaran dengan pengecekan input menggunakan JavaScript.

- **Tampilan Responsive**  
  Layout tetap nyaman dilihat di layar laptop maupun smartphone.

- **Manipulasi DOM**  
  Konten halaman dapat diubah secara dinamis melalui JavaScript tanpa reload.

---

## 🛠️ Teknologi yang Digunakan

- **HTML5** → Membuat struktur halaman website  
- **CSS3** → Mengatur desain dan tampilan visual  
- **JavaScript (ES6)** → Menambahkan logika dan interaksi pengguna
